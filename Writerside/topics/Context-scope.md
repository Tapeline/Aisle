# Context scope

Context scope is defined as follows:
```
scope context <project name>
```

Then context entities declarations follow:

Actor:
```
actor <name>
```
or
```
actor <name>:
    [<text description>]
    [links block]
```

System:
```
[external] system <name>
```
or
```
[external] system <name>:
    [<text description>]
    [links block]
```
